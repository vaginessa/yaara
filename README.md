# Yaara #
[![Build Status](https://travis-ci.org/Mlieou/yaara.svg?branch=develop)](https://travis-ci.org/Mlieou/yaara)

Yaara is an Aria2 Remote client for Android

---

## Features
- Multiple Aria2 server support
- Real-time task status
